Current Projects

    • Exploiting Criticality using Hardware Prefetching and Branch Prediction Techniques
      Implementing Criticality Aware Tiered Cache Hierarchy. Understanding other Memory Technologies and integrate them within cache to gain better performance. Performing experiments on Branch predictors to optimise the use of criticality on different sizes of cache.
    • Design and Development of Runtime Monitor Processors
      Understanding the concepts of Runtime Verification and Monitorability and real-time Realizable, Responsive, Unobstrusive Unit (rt-R2U2). Implementing Temporal-Logic Based Runtime Observer Pairs for System Health Management(SHM) of Real-Time Systems. Implementation of FSM models(monitor processor) using Haskell on CLASH compiler.
    • Analysing cloud workloads to improve energy efficiency
      Understanding the features and services provided by IBM cloud. Profiling of applications based on benchmark workloads. Comparative study of different storage services.
